# San Diego Youth GAA Website

A modern, responsive website for San Diego Youth GAA - teaching Gaelic Football and Hurling to children ages 4-14.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Accessibility**: Keyboard navigation and screen reader friendly
- **Fast Loading**: Optimized for performance
- **SEO Ready**: Proper meta tags and semantic HTML structure

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About**: Information about the club and its mission
3. **Sports**: Detailed explanations of Gaelic Football and Hurling
4. **Training Methodology**: Age-appropriate development approach
5. **Sessions**: Upcoming training sessions and registration info
6. **Sponsorship**: Opportunities for community support
7. **Contact**: Contact form and information

## Setup for GitHub Pages

### 1. Create GitHub Repository

1. Go to [GitHub](https://github.com) and create a new repository
2. Name it `sdyouthgaa` (or your preferred name)
3. Make sure it's set to **Public** (required for free GitHub Pages)
4. Initialize with README (optional)

### 2. Upload Files

1. Upload all files (`index.html`, `styles.css`, `script.js`) to the repository
2. Commit and push to the main branch

### 3. Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on **Settings** tab
3. Scroll down to **Pages** section
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**

### 4. Access Your Website

Your website will be available at:
`https://yourusername.github.io/sdyouthgaa`

## Customization

### Content Updates

To update content, simply edit the HTML file:

- **Club Information**: Update the hero section and about section
- **Session Details**: Modify the sessions section with current dates/times
- **Contact Info**: Update email, phone, and location details
- **Sponsorship**: Adjust sponsorship levels and benefits

### Styling Changes

The CSS file (`styles.css`) contains all styling. Key sections:

- **Colors**: Update the color scheme by changing CSS variables
- **Fonts**: Modify the Google Fonts import and font-family declarations
- **Layout**: Adjust grid layouts and spacing as needed

### Adding Images

1. Create an `images` folder in your repository
2. Upload your images
3. Update the HTML to reference the images:
   ```html
   <img src="images/your-image.jpg" alt="Description">
   ```
